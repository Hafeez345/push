# AI Agent for Data Queries

## Description
This AI Agent is built in Python. It can answer all types of questions based on the data you provide. You input your data, and the AI Agent analyzes it to generate accurate responses.

## Features
- Reads the input data  
- Provides answers based on the data  
- Supports custom filters and queries  
- Displays output in the terminal (future support for Excel/CSV export possible)
